AI Marketing Assistant
Небольшое веб-приложение на Flask с использованием OpenAI API для генерации маркетинговых описаний товаров.

Возможности:
описание товара,преимущества,слоганы,хештеги,кнопка Copy и Loader

Технологии:
Python, Flask, OpenAI API, HTML/CSS, JavaScript

Запуск:
pip install -r requirements.txt
python app.py

Создать .env:
OPENAI_API_KEY=your_key

Открыть в браузере:
http://127.0.0.1:5000
